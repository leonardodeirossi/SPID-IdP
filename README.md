# Identity Provider SPID
## IdP SPID compatibile con lo standard OAuth 2.0

SPID-IdP è un Identity Provider SPID compatibile con lo standard OAuth 2.0.

È stato realizzato secondo le specifice AgID e implementa il set di scope/claim completi (Determinazione AgID n. 160 del 2018 recante «Regolamento per l’adozione di linee guida per l’attuazione del Codice dell’Amministrazione Digitale»).
